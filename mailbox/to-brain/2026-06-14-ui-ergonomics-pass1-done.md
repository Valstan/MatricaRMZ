## Сделано (объективные)
- **#1 Контрагент UUID→имя** + **#5 dev-жаргон вкладок** — PR #365 (прошлая сессия).
- **#2 левый отступ списков** («таблица уехала вправо»). Корень: `table.list-table { margin-inline: auto }` центрировал узкую таблицу в широком контейнере. Фикс: `margin-inline: 0` (left-align) — системно во всех списках. **Before/after (CDP, список двигателей):** leftOffset 407px (центр) → **1px** (flush-left, заполняет колонку).
- **Обрезанные заголовки** «Дата прихода»→«Дата прих…», «Доступно»→«Досту…». Фикс: `table.list-table th { white-space: normal; overflow-wrap: anywhere }` — переносятся, не «…». **Before/after:** `whiteSpace nowrap`+clipped → `normal`, `clipped: false`. (Ячейки данных остаются nowrap+ellipsis.)
- **#6 омоним-кнопки топ-бара** — «Отправить в чат»→**«Ссылку в чат»**, «Сохранить в заметки»→**«Ссылку в заметки»** (объект «ссылка [на раздел]» вынесен вперёд, destination различает). Подтверждено владельцем.
