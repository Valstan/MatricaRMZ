# Т-13 — выполнено (Ф1+Ф2), Фаза 3 снята владельцем

## Что сделано (в `main`, ждёт ближайшего релиза)

DB-backed модуль «Табель учёта рабочего времени» по цехам (PR #453 backend, #454 UI, #455 печать+комментарии):

- **Данные:** `timesheets`/`timesheet_rows`/`timesheet_cells`(код+часы+comment)/`timesheet_codes`(справочник, засев 19 кодов Т-13). Аддитивные миграции 0062/0063, серверный API-модуль (без client-SQLite-sync). Реестр сотрудников/цехов переиспользован.
- **Ввод (главное требование «щёлкнул — вставилось»):** палитра чипов (коды+часы) клик-вставка + «кисть»; Excel-клавиатура (цифры→часы, буква→код по префиксу, стрелки/Enter/Tab, Del, Ctrl+D); «Заполнить норму всем»; все дни месяца, воскресенья приглушены но редактируемы (правка владельца соблюдена).
- **Live-подсчёты:** Σ/строка, явка/день, итог vs норма (×чел).
- **Печать:** модалка-галочки — Первая половина (1–15) / Вторая (16–N) / Месяц целиком / Расшифровки по работникам, любая комбинация → A4 + PDF, легенда кодов.
- **Расшифровки (пожелание владельца):** комментарий «где был/что делал» в ячейке (работник×день) → на печати отдельная страница по работникам.
- Каждый этап под зелёными гейтами (typecheck/lint/backend-test 261) + CDP-verify живого UI.

## Фаза 3 — СНЯТА (решение владельца 2026-06-17)

Производственный календарь РФ (праздники) + привязка к зарплате — **не делаем**. Дословно: частное предприятие, праздники «не особо соблюдаются»; зарплата «пока в программе не особо начисляется» — у сдельщиков по нарядам, считается отдельным человеком, который в программе не работает. Если понадобится — отдельная масштабная разработка (своя нитка/директива).

## Open перед стройкой — как разрешились

- Точные числовые коды Т-13 — засеяли стандартный Госкомстатовский набор, справочник редактируемый; HR при необходимости поправит.
- `0`/`Н` старого Word — не нужно: старые данные не импортируем (ведём с чистого листа).
- 6/5-дневка — конфиг на цех, дефолт 6 (как цех №2).
- Ростер — реестр сотрудников ЭРП (picker), снапшот должности в строке.
- Привязка к зарплате — снята (см. Фаза 3).
- `.docx`-референс печати был на машине, разобран; печать сделана по структуре (№/ФИО/дни с днём недели/итоги/легенда/подписи-блок).
