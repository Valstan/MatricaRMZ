# Inbox надиктовок владельца

Сюда кидаются **сырые** мысли/надиктовки как есть — без форматирования, любым файлом (`YYYY-MM-DD-<пара-слов>.md` или просто `.txt`). Ничего оформлять не нужно.

`/zavod`-сессия разбирает inbox: превращает надиктовку в оформленную идею (`../ideas/NNN-<slug>.md` + строка в `../INDEX.md`) и удаляет исходник из inbox (история остаётся в git).
