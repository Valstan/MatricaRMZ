# 006 — Виды работ (справочник + нормо-часы) и мотивация бригад

**Статус:** 💬 обсуждается · обновлено 2026-06-24

## Проблема
1. **Справочник:** виды работ называются в цехах и ПЭО по-разному; цеха на сдельщине (1 — сдельно, 3 — смешанно, 4 — повременка), ПЭО — по часовым тарифам, конвертируют вручную.
2. **Мотивация (надиктовка 24.06):** рабочие сознательно делают 3 вместо 6 в день — боятся, что «за те же деньги заставят делать 6» и что кончится работа. Завод не может платить повременно (по часам обязан оплатить простой, а сырьё не гарантировано) → отсюда сдельщина. За непрошедшую обкатку сейчас штрафуют. Бригадиров «за узел» не назначили — не с кого спросить.

## Варианты / решение
- **Справочник:** одно название работы для цеха и ПЭО + у каждой работы сдельная расценка И нормо-часы (как унификация номенклатуры, идея 001). Сначала собрать с цехами и ПЭО (ещё не начато).
- **Мотивация (к обсуждению):** гарантированный минимум (снять страх простоя) + сдельный бонус за двигатель + **премия за качество потока** (обкатка с первого раза) ВМЕСТО штрафа; узловая ответственность бригадира (отвечает и поощряется за узел).

## Мировая практика
Gainsharing, премия за first-pass-yield (качество с первого раза) вместо штрафа, ячеистая (бригадная) ответственность и оплата за узел, гарантированный минимум при нестабильной загрузке. Оговорка: схему утверждает директор; усложнять расчёт нельзя — иначе рабочие не поймут и не поверят.

## Рекомендация (по делегированию владельца, 2026-06-24)
Разделить программное и денежное:
- **Программная часть (нейтральна к политике) — делать:** поле «нормо-часы» у вида работ + отчёт ПЭО + видимость бригаде суммы по наряду (прозрачность снимает страх «обманут на норме»).
- **Денежная схема** (премия за первопроходную обкатку вместо штрафа, гарантированный минимум, оплата за узел) — **рекомендую, но это решение директора**; передать ему отдельно.
- **Старт без кода:** собрать единый справочник видов работ с ПЭО и цехами (по образцу номенклатуры, идея 001).

## Заземление
- **Уже есть:** наряды (4 типа) с подписями и ценообразованием, связь с зарплатой исполнителей, табель Т-13 (учёт часов).
- **Легко доделать:** поле «нормо-часы» у вида работ; отчёт ПЭО «наряды в нормо-часах за период»; видимость бригаде суммы по наряду.
- **Месяц:** автоматический расчёт зарплаты из нарядов+табеля; помесячный подсчёт премии за первопроходную обкатку.

## → в программу (после сбора справочника)
Поле «нормо-часы» у услуги/вида работ + отчёт ПЭО в нормо-часах (естественное расширение нарядов). Мотивационные схемы — после утверждения директором; начать с прозрачности (показывать бригаде сумму по наряду).
