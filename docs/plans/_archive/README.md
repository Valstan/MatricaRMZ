# Архив планов

Сюда переезжают **завершённые** многоэтапные планы из [`docs/plans/`](../), чтобы активная папка планов держала только то, что в работе. Каждый архивный план индексируется строкой в [`docs/COMPLETED.md`](../../COMPLETED.md).

Перемещать через `git mv docs/plans/<slug>.md docs/plans/_archive/<slug>.md` (история сохраняется). Делает `/close_session` при закрытии нитки; массовый первичный слив — PR-2 плана [`memory-reorg-2026-06.md`](memory-reorg-2026-06.md).

> Внутренние относительные ссылки в архивных планах (`../file`) после переезда на уровень глубже могут указывать на сдвинутый путь — это исторические документы, чинить ссылки в них не нужно (granular-привязки — по `file:line` в `git log`/PR).
