# Глобальный паспорт окна Matrica RMZ

## Идентификация
- Контур: Windows Electron client
- Главный compose: `electron-app/src/renderer/src/ui/App.tsx`
- Навигация: `electron-app/src/renderer/src/ui/layout/Tabs.tsx`

## Каноническая структура окна
1. Верхний уровень приложения (`App`): состояние авторизации, текущая вкладка, контекст карточек.
2. Левая/верхняя навигация по департаментам (группы табов).
3. Область контента (list/detail/report/card) с адаптивным layout.
4. Модальные/служебные слои: sync progress, release welcome, fatal dialogs.

## UX-инварианты
- Переход list -> card и обратно должен сохранять контекст.
- Карточки не «ломают» общий ритм, используют `EntityCardShell`/`SectionCard`.
- Числовые данные выравниваются вправо, вторичный текст — через `ui-muted`.
- На узких окнах layout деградирует в 1 колонку без горизонтального хаоса.

## Визуальные токены
- Основной источник: `electron-app/src/renderer/src/ui/global.css`
- Базовые классы: `entity-card-grid`, `entity-card-span-full`, `ui-table`, `ui-muted`
- Стандарт: `docs/UI_VISUAL_STANDARDS.md`

## Source map
- Состояние окна и роутинг: `App.tsx`
- Группы меню/подписи/дефолтный порядок: `Tabs.tsx`
- Глобальные переменные и UI-ритм: `global.css`
